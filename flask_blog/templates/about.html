About Page

This Blog Management System is a backend-driven web application built to manage content creation and publishing workflows efficiently. The platform allows authenticated users to create, edit, and manage blog posts while maintaining structured role-based access.

The system is designed with scalability and security in mind, implementing secure authentication mechanisms and database-backed content storage.

Core Features

  • User authentication and authorization
  • Role Based Access Control
  • Create, update, and delete blog posts
  • for blog posts
  • Database integration

Tech Stack

Python, Flask, SQLAlchemy, PostgreSQL, REST APIs, and production deployment using Gunicorn.

This project was built to deepen my understanding of backend system design, authentication workflows, and scalable web application architecture.
